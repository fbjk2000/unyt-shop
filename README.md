# UNYT.shop

Premium marketing site and app shell for the UNYT ecosystem, built with Next.js App Router, TypeScript, Tailwind CSS, and Framer Motion.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the local dev server:

```bash
npm run dev
```

3. Validate the project:

```bash
npm run lint
npm run typecheck
npm run build
```

## Routes

Public routes:
- `/`
- `/ecosystem`
- `/how-it-works`
- `/security`
- `/faq`

App routes:
- `/app/wallet`
- `/app/swap`
- `/app/activity`
- `/app/settings`

## Notes

- All balance, wallet, swap, and ecosystem data is static mock content for v1.
- Real wallet, swap, payments, auth, and partner integrations are intentionally deferred.
