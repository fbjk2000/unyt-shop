## Design usage

- The original multicolour lockup remains the primary logo reference.
- Derived motifs intentionally bias toward the existing site blue so the vNext iteration preserves the current visual anchor.
- Pink from the source logo is used as a small secondary accent for corners, dividers, and micro-details rather than as a dominant page colour.
