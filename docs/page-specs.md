# Page specs for unyt.shop

## Home
Goal: explain value in one screen and drive wallet creation or ecosystem exploration.

Required sections:
- Announcement bar
- Premium header / nav
- Hero with strong copy and product UI
- Ecosystem proof strip with Fintery, Alakai, TechSelec, Unyted.world, Unyted.Chat, Earnrm.com
- 3-step explainer: Add UNYTs / Spend across products / Move on-chain when available
- Feature section: one balance across brands
- Feature section: mobile-first checkout and top-up
- Feature section: swap clarity
- Trust band
- FAQ preview
- Closing CTA
- Premium footer

## Ecosystem
Goal: show where UNYTs work.

Required sections:
- Hero with search/filter shell
- Featured ecosystem cards
- Detail blocks for each product
- How spending works
- Partner CTA

## Wallet
Goal: make balance, actions, and spending clear.

Required sections:
- App shell header
- Large balance card
- Quick actions: Add funds / Pay with UNYTs / Withdraw / Swap
- Spend destinations
- Activity feed preview
- Payment method and support modules

## Swap
Goal: make conversion and movement feel precise and trustworthy.

Required sections:
- Intro copy
- Swap form card
- Quote details panel
- Popular pairs
- Recent activity
- Safety and availability panel

## Security
Goal: create confidence and maturity.

Required sections:
- Security hero
- Principles grid
- Wallet and transaction safety notes
- Support and account recovery guidance
- Risk / availability disclosure

## FAQ
Goal: answer the main product questions simply.

Required questions:
- What are UNYTs?
- Where can I use UNYTs?
- How do I get UNYTs?
- Can I pay on mobile?
- Are wallet features available everywhere?
- How does swap work?
- What products accept UNYTs?
- How do I get support?
